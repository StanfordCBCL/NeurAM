# NeurAM: Nonlinear dimensionality reduction for uncertainty quantification through neural active manifolds

## Usage

```
neuram <path_to_input.json>
```

## Inputs

```json
{
  "firstName": "John",
  "lastName": "Smith",
  "age": 25
}
```
